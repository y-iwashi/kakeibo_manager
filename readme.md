# 家計簿DB管理アプリ  
(サードパーティ製アプリ)  

## フロント
HTML5 / CSS3 / JavaScript (ES6+)

特徴 :  
CSS Grid / Flexbox: 検索パネルの2カラムレイアウトやアニメーションを実現。  
Fetch API: 画面をリロードせずにサーバーとデータをやり取り（非同期通信）。

## バックエンド
実行環境 : Node.js

## コマンド集（備忘録）
### サーバーの起動  
```
node server.js
```
